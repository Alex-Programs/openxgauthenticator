Free, open source Sophos XG Authenticator

Login to Sophos XG firewalls with a single command.

  • Open source - Github Repository
  • Written in Rust for safety, reliability, low memory usage, easy setup, and speed
  • Configuration options for retry time, keepalive time, and whether to lie about the user-agent
  • Cross platform: Windows, Linux, MacOS
  • Local: It doesn't communicate with anything but the firewall. No data exfiltration - just check Wireshark.

Usage

To use, simply run the executable and enter username + password when prompted. Credentials will be saved in config.toml for next time. Configuration options are also in config.toml. You can open it in any text editor.
